**Goal:** Replace the generic JSON playground with an agent section built from four real
artifacts: the Button manifest, per-client MCP install commands, an example prompt, and a
CascadeView demo that renders a slice of the Relay console from visible JSON. Refresh
QuickStart (anchor) and close the page with a full-width CTA band.

**Architecture:** One section file `src/sections/AgentLayer.tsx` with a two-column artifact
grid; manifest source imported at build time via `?raw` (string, zero runtime cost); the
CascadeView slice reuses Relay fixture data so the JSON pane and the console tell one story.

**Tech stack:** Vite `?raw` imports, `@cascade-ui/render` CascadeView, cascade Tabs +
CopyCommand.
