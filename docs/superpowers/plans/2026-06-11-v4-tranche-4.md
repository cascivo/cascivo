**Goal:** Upgrade AppShell so a console actually works end to end: a `createShellState()` signals
object the header can drive (hamburger → drawer/collapse), a collapsible right aside, a mobile
drawer with scrim, and the skip-link target. Then ship `block/console-app` — the Carbon-parity
dogfood block composing ShellHeader + SideNav + HeaderPanel + Switcher + aside + content.

**Architecture:** `ShellState` is a plain object of signals + methods created by
`createShellState()` (module function, callable anywhere — component body via lazy `useSignal`
init, or app module scope). AppShell accepts `state?: ShellState`; when omitted it creates one
internally with the `persistKey` props (back-compat). CSS owns all responsive switching via one
`64rem` media query; JS never reads viewport size except `toggleSideNav()`'s `matchMedia` check
at interaction time (master plan decision 4, edge case 9).

**Tech stack / Conventions:** identical to T1–T3.

```ts
import { signal, type Signal } from '@cascade-ui/core'
import { persistedSignal } from '@cascade-ui/storage'

export interface ShellState {
  /** Desktop: side nav collapsed to the icon rail. Persisted. */
  sideNavCollapsed: Signal<boolean>
  /** Mobile: side nav drawer open. Never persisted. */
  sideNavOpen: Signal<boolean>
  /** Right aside visible. Persisted. */
  asideOpen: Signal<boolean>
  /** Hamburger handler: drawer on mobile, collapse on desktop (resolved at call time). */
  toggleSideNav: () => void
  toggleAside: () => void
}

export interface CreateShellStateOptions {
  /** Storage key prefix; false disables persistence. Default: 'cascade.shell'. */
  persistKey?: string | false
}

const DESKTOP = '(min-width: 64rem)'

function isDesktop(): boolean {
  // SSR: assume desktop — the drawer is a client-only interaction (edge case 9)
  if (typeof window === 'undefined' || typeof window.matchMedia !== 'function') return true
  return window.matchMedia(DESKTOP).matches
}

export function createShellState(options: CreateShellStateOptions = {}): ShellState {
  const key = options.persistKey ?? 'cascade.shell'
  const sideNavCollapsed = key === false ? signal(false) : persistedSignal(`${key}.sidenav`, false)
  const asideOpen = key === false ? signal(true) : persistedSignal(`${key}.aside`, true)
  const sideNavOpen = signal(false)

  return {
    sideNavCollapsed,
    sideNavOpen,
    asideOpen,
    toggleSideNav: () => {
      if (isDesktop()) {
        sideNavCollapsed.value = !sideNavCollapsed.value
      } else {
        sideNavOpen.value = !sideNavOpen.value
      }
    },
    toggleAside: () => {
      asideOpen.value = !asideOpen.value
    },
  }
}
```

Implementation notes:

- Lazy-init: keep the existing `useSignal<...>(null)` lazy-store pattern, but store a full
  `ShellState`: `if (store.peek() === null) store.value = state ?? createShellState({ persistKey })`.
  When `state` is provided it wins; the old `persistKey`-only path maps onto
  `createShellState({ persistKey })` — delete the now-redundant inline collapse logic (your
  change makes it unused; removing it is in scope).
- Aside: `<div className={styles['aside']} data-state={shell.asideOpen.value ? 'open' : 'closed'}>` —
  CSS animates `grid-template-columns` participation via `inline-size` between
  `var(--cascivo-shell-aside-inline-size)` and `0` with `overflow: hidden` + content
  `visibility: hidden` when closed (so `not.toBeVisible()` passes).
- Drawer state on the root: `data-drawer={shell.sideNavOpen.value ? 'open' : undefined}` (consumed
  in Task 3).
- Main: `<main id="cascade-main" tabIndex={-1} className={styles['main']}>`.
- The internal collapse toggle button: keep it for back-compat but it now calls
  `shell.toggleSideNav()`.
- i18n: add `aside` labels if the aside gets its own toggle affordance; the console block uses a
  ShellHeader action instead, so only add strings if you render a toggle (proposal: no built-in
  aside toggle button — the header drives it; skip new strings).

- Escape: a `useSignalEffect` that attaches a document `keydown` listener only while

```tsx
useSignalEffect(() => {
  if (!shell.sideNavOpen.value) return
  const onKey = (e: KeyboardEvent) => {
    if (e.key === 'Escape') shell.sideNavOpen.value = false
  }
  document.addEventListener('keydown', onKey)
  return () => document.removeEventListener('keydown', onKey)
})
```
