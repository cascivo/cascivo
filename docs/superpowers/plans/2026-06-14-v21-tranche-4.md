> backbone — issues are owned in storage, so all mutations survive reload by construction.
>
> Current-state assumptions — RE-VERIFY before starting:
>
> - T1 kit + reference patterns shipped. `@cascivo/react` exports CommandMenu, Kbd, SegmentedControl,
>   Drawer, Form, MultiSelect, Combobox, ContextMenu, DataTable, Badge, Avatar (re-verify names).
> - `@cascivo/storage` exposes persisted-signal helpers (re-verify the exact API used in T1's AppShell
>   theme persistence and reuse the same pattern).

**Goal:** A functional **Cascade Track** issue tracker: a board view (status columns) ↔ list view
(`DataTable`) toggled by `SegmentedControl`; a Cmd+K `CommandMenu` and `Kbd` shortcuts to create/move/
assign issues without a mouse; an issue create/edit `Drawer` with a `Form`, `MultiSelect` (labels), and
`Combobox` (assignee); right-click `ContextMenu` row actions. Issues live in `@cascivo/storage`, so every
mutation persists across reload.

**Architecture:** New `apps/examples/track/` mirroring T1 (vite config + aliases incl.
`@cascivo/example-kit`, `data/` seed, `store/` persisted signals, `App.tsx`, `sections/`, `test/`). The
mock-api here is thin: seed data hydrates the persisted store on first run; thereafter the store is the
source of truth.

**Tech stack:** unchanged. `useSignals()` first; `useSignalEffect` for keyboard listeners (NOT
`useEffect`); no `useState`/`useContext`. Strings via `@cascivo/i18n`.

## Tranche exit criteria

1. `apps/examples/track` runs and builds with no prior full build.
2. Issues live in a persisted (`@cascivo/storage`) store seeded on first run; every mutation (create,
   move, edit, assign, label, delete) survives reload.
3. Board ↔ List toggle works and persists; filtering uses a derived (`useComputed`) signal.
4. Cmd+K command menu + global keyboard shortcuts work (registered via `useSignalEffect`, not
   `useEffect`) and don't fire while typing.
5. Playwright smoke + axe pass; no `useState`/`useEffect`/`useContext`; `useSignals()` first; strings via
   `@cascivo/i18n`; mobile-clean 320–414.
