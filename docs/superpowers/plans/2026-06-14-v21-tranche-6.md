## Tranche exit criteria

1. An examples gallery lists all five apps (codename, "feels like", library slice, live link) and is
   linked from docs/landing; it states the mock/no-backend nature.
2. Each app + the kit has a README documenting it as a mock demo; each app shows the in-app mock
   affordance.
3. A deterministic coverage test passes, asserting no top-tier `@cascivo/react` component and no
   `@cascivo/charts` chart is entirely undemoed (gaps closed or explicitly allowlisted).
4. `cf-pages.yml` has a paths-filtered deploy job per app; all five build with no prior full build; CI
   `verify` builds/tests them.
5. Full CLAUDE.md gate exits 0; all five apps verified functional offline; all smoke + axe specs pass.
6. `ROADMAP-V21.md` DoD all checked; status → ✅ Shipped.
