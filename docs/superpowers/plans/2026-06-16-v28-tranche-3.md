# v28 — Tranche 3: Control Panel

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the left-column control panel: preset tiles, base mode toggle, hue/chroma sliders
with live swatches, a discrete radius slider, a font `Select`, and a reset button. All controls
write to the `config` signal from T2.

**Architecture:** `ControlPanel.tsx` imports `config` from `store.ts`, writes to `config.value`
directly. It uses cascivo components (`Toggle`, `Select`, `Button`, `Tooltip`) for interactive
controls and plain `<input type="range">` for the sliders (cascivo has no Slider component yet).
All reads of `config.value` in the render function require `useSignals()` at the top.

**Tech Stack:** `@cascivo/core` (useSignals), `@cascivo/react` (Toggle, Select, Button, Tooltip),
React 18, plain `<input type="range">`, CSS custom properties for the swatch colors.

- [ ] **Step 2: Create `apps/landing/src/pages/create/ControlPanel.tsx` with preset tiles only**

  ```tsx
  'use client'
  import { useSignals } from '@cascivo/core'
  import { Button } from '@cascivo/components/button'
  import { config, DEFAULT_CONFIG } from './store'
  import { PRESETS } from './presets'

  export function ControlPanel() {
    useSignals()
    const current = config.value

    function applyPreset(id: string) {
      const preset = PRESETS.find((p) => p.id === id)
      if (preset) config.value = preset.config
    }

    return (
      <div className="ctrl-panel">
        {/* Preset tiles */}
        <section className="ctrl-section">
          <h3 className="ctrl-label">Presets</h3>
          <div className="ctrl-presets" role="listbox" aria-label="Theme presets">
            {PRESETS.map((preset) => (
              <button
                key={preset.id}
                type="button"
                className="ctrl-preset-tile"
                role="option"
                aria-selected={current.presetId === preset.id}
                data-state={current.presetId === preset.id ? 'active' : undefined}
                onClick={() => applyPreset(preset.id)}
              >
                <span
                  className="ctrl-preset-swatch"
                  style={
                    {
                      '--swatch-bg': preset.swatchBg,
                      '--swatch-accent': preset.swatchAccent,
                    } as React.CSSProperties
                  }
                  aria-hidden="true"
                />
                <span className="ctrl-preset-label">{preset.label}</span>
              </button>
            ))}
          </div>
        </section>

        {/* Reset */}
        <Button
          variant="ghost"
          size="sm"
          onClick={() => {
            config.value = DEFAULT_CONFIG
          }}
          className="ctrl-reset"
        >
          Reset to default
        </Button>
      </div>
    )
  }
  ```

### Task 3: Radius slider and font Select

- [ ] **Step 1: Add radius section to `ControlPanel.tsx`**

  Import `RADIUS_STOPS` and `RADIUS_LABELS` from `./store`. Append in `<div className="ctrl-panel">`:

  ```tsx
  import { config, DEFAULT_CONFIG, RADIUS_STOPS, RADIUS_LABELS } from './store'
  import type { RadiusStop } from './store'

  {
    /* Radius */
  }
  ;<section className="ctrl-section">
    <h3 className="ctrl-label">Border radius</h3>
    <div className="ctrl-radius-track">
      <input
        type="range"
        min={0}
        max={RADIUS_STOPS.length - 1}
        step={1}
        value={RADIUS_STOPS.indexOf(current.radiusBase)}
        onChange={(e) => {
          const stop = RADIUS_STOPS[Number(e.target.value)] as RadiusStop
          config.value = { ...current, radiusBase: stop, presetId: null }
        }}
        aria-label="Border radius"
        aria-valuetext={RADIUS_LABELS[current.radiusBase]}
      />
      <div className="ctrl-radius-ticks" aria-hidden="true">
        {RADIUS_STOPS.map((stop) => (
          <span key={stop} className="ctrl-radius-tick">
            {RADIUS_LABELS[stop]}
          </span>
        ))}
      </div>
    </div>
  </section>
  ```
