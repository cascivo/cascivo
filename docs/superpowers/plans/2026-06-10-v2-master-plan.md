# cascade v2 — Master Plan (Roadmap v2 execution sequencing)

> **For agentic workers:** This is a sequencing document, not an executable task plan. Execute the
> per-tranche plans listed below (Tranche 1 exists; later tranches get their detailed plan written
> just-in-time with the `superpowers:writing-plans` skill when the previous tranche completes).
> Source spec: `docs/ROADMAP-V2.md`.

**Goal:** Implement Roadmap v2 — from component library to application platform — in seven
sequential tranches, each producing working, gated, reviewable software.

**Architecture:** Components flow through the existing dark-factory pipeline
(`factory-backlog.json` + `/factory` skill). New platform packages (`i18n`, `storage`, `charts`,
`render`) are hand-planned npm packages. Generated artifacts (registry, READMEs, schema, MCP data)
are protected by a fail-on-drift CI gate from Tranche 1 onward.

**Tech stack:** pnpm workspaces + vite+ (`vp`), TypeScript strict, React 18+ with
`@preact/signals-react`, CSS Modules + `@layer cascade.*`, Vitest + Testing Library + Playwright.

---

## Decisions locked in (2026-06-10, with Adam)

1. **Plan shape:** master doc + detailed tranche plans written just-in-time (this file + one plan per tranche).
2. **Drift gate:** CI **fails on PR** when generated artifacts drift; the `registry.yml` auto-commit workflow is removed. Generated artifacts ship in the same PR as their source change.
3. **Layouts distribution:** copy-paste registry items under a new `layout` registry category (not an npm package).
4. **First-party locales:** `en` + `de` only for built-in component strings; more locales demand-driven.
5. (From roadmap header) Charts fully from scratch; JSON UI runtime renderer first, codegen second; pragmatic component parity (~35–40); i18n = typed catalogs + native `Intl`.

## Exit criteria per tranche (verbatim from roadmap where defined)

- **T1:** registry URLs correct everywhere; drift gate red on un-regenerated PRs; animation audit green; all ≤34-priority backlog items `review`/`done`; swept components render German strings under `set('de')` (component-test verified — a user-facing docs locale switcher lands with the docs overhaul in T6); docs theme survives reload through `persistedSignal` with no flash.
- **T2:** 10,000-row docs demo — render/scroll/sort/filter feel identical to 10 rows, enforced by Playwright latency budgets in CI; re-render contract asserted via probe (0 commits outside the table, ≤1 table commit per interaction — see edge case 14); cmd-k is docs search.
- **T3:** `npx cascade add layout/app-shell block/users-table-page` produces a running page; docs + landing dogfood the layouts.
- **T4:** 5,000-point line chart with smooth hover; visual snapshots in all three themes; dashboard block gains real charts.
- **T5:** an MCP-connected agent produces a validated JSON config rendering live, without writing TSX; codegen round-trip test green in CI.
- **T6:** all Definition-of-Done gates from ROADMAP-V2 enforced in CI; perf comparison page live.
