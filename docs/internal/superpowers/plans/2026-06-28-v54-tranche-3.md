# v54 — Tranche 3: Caret-Anchored Command Menu

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Render the command menu and position it **at the caret** (S-3/S-6) — the visible half of the feature. Lens:
**ProseMirror/TipTap + `@cascivo/core`.** The challenge: `useAnchorPosition` (`core/anchor.tsx:102`) needs a **DOM**
`anchorRef`, but a textarea caret is not a DOM node (`anchor.tsx:10-16`). Solve it the way ProseMirror does with
floating-ui — a **virtual element**: render a 0×0 "caret-proxy" `<div>` at the T1 `caretCoords` and feed *that* as the
anchor. The menu itself mirrors the `FindPanel` overlay conventions (`find-panel.tsx` + `find-panel.module.css`: absolute
in `.codeArea`, themed via `--cascivo-editor-*`, `z-index: 2`, ≥44px coarse) but anchored at the caret rather than the
top-right corner.

**Architecture:** a menu component + a caret-proxy + the anchor bridge.
- **`slash-menu.tsx`**: a controlled, signal-safe `role="listbox"` of `option`s — `{ commands, activeIndex, onSelect,
  onHover, query, labels }`. Each option has a stable `id` (e.g. `slash-opt-${i}`) for `aria-activedescendant` (T4),
  a label + optional hint, highlights the active row, and shows an i18n empty-state when the list is empty. No internal
  state; the parent owns `slashIndex`.
- **Caret-proxy**: a 0×0 absolutely-positioned `<div>` in `.codeArea`, its `top/left` set (imperatively, in a
  `useSignalEffect`, or via a style signal) to the T1 `caretCoords` of the live caret. It carries the
  `useAnchorPosition` `anchorStyle`.
- **Bridge**: `const { anchorStyle, floatingStyle } = useAnchorPosition({ anchorRef: caretProxyRef, floatingRef:
  menuRef, placement: 'bottom-start', enabled: slashOpen })`. Spread `anchorStyle` on the proxy, `floatingStyle` on the
  menu. The hook's CSS-anchor path handles scroll natively; its JS fallback recomputes on scroll/resize.

**Tech Stack:** `slash-menu.tsx` + `slash-menu.module.css`; `useAnchorPosition` from `@cascivo/core`; the T1 `caret.ts`;
`useSignalEffect`/`useRef` for the proxy. No new dependency.
