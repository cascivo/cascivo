# v36 — Tranche 4: Signature Showcase (Consolidation)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the five separate home demo sections with **one** signature showcase: a single
`Tabs` (or `SegmentedControl`) container that holds a small set of demo views. The default view is the
anchor demo (`ThemeDemo`); secondary tabs are compact teasers (charts, relay) that deep-link to their
full versions on `/examples`. This is the "one interactive moment" of the minimal home, and it dogfoods
cascivo's own `Tabs`.

**Architecture:** New `apps/landing/src/sections/Showcase.tsx` composes `@cascivo/components/tabs`
(or `segmented-control`). The default tab renders the existing `ThemeDemo`; non-default tabs are lazy
(`React.lazy` + `Suspense`) so they don't ship in the eager home bundle. `HomePage` renders `Showcase`
in the slot that T2 filled with `ThemeDemo`.

**Tech Stack:** React 18 island, `@cascivo/core` signals (`useSignals` first in any signal-reading
component), `@cascivo/components` `tabs`/`segmented-control`, vite+ (`vp`).
