# v36 — Tranche 5: Progressive Disclosure + Nav De-clutter

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give curious visitors a way to expand depth **in place** without re-bloating the home, and
slim the top nav to match the minimal page. Add `Collapsible` "Learn more" expanders to the Principles
cards and the Quick Start "prebuilt option", and group the secondary nav links under a "Resources"
`NavigationMenu`. Dogfoods cascivo's own `Collapsible`/`Accordion` and `NavigationMenu`.

**Architecture:** Edits `Principles.tsx`, `QuickStart.tsx`, and `Header.tsx`, plus `landing.css`. All
disclosure uses `@cascivo/components` (`collapsible`, or `accordion` for a grouped list). Disclosure
panels stay in the a11y tree and keyboard-reachable — no `display:none`-only hiding.

**Tech Stack:** React 18 island, `@cascivo/core` signals, `@cascivo/components`
(`collapsible`/`accordion`, `navigation-menu`/`menu`), vite+ (`vp`).
