# v36 — Tranche 1: Content Relocation (Zero Loss)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give every home section that is leaving the home page a verified destination **before** the
home is trimmed in T2. Build the new `/ai` page, move `SignalsDemo`/`ProofTeasers` onto `/performance`,
`ChartShowcase` (+ relay/examples teasers) onto `/examples`, and `Ecosystem` onto `/guides` and the
footer. Wire routes, nav, SEO, search index, and sitemap. **The home page is unchanged in T1** — this
tranche only adds destinations.

**Architecture:** Existing section components (`apps/landing/src/sections/*`) are reused as-is; new
page components (`apps/landing/src/pages/*`) host them with page chrome. Routing is the `ROUTES` map in
`App.tsx` + `ROUTE_HEAD`/`seo.ts`; search is `search/buildIndex.ts`. No demo internals change.

**Tech Stack:** React 18 island, `@cascivo/core` signals, vite+ (`vp`).
