# DashboardCharts

Dashboard layout with KPI tiles, line chart, bar chart, and pie chart.

## Install

```bash
npx cascade add block/dashboard-charts
```

## Category

`display`

## Props

_No props._

## Examples

### Default

Charts dashboard demo

```tsx
<DashboardCharts />
```

## Accessibility

- **WCAG level:** AA
- **ARIA role:** `generic`

## Dependencies

- `@cascade-ui/charts`
- `@cascade-ui/react`

## Tags

block, dashboard, charts, kpi
