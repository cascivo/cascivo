**Goal:** Every CSS custom property is `--cascivo-*`. Zero `--cascade-` references remain anywhere
in source (CSS, TS/TSX, JSON, test fixtures, snapshots). The `--cascivo-brand-*` tokens from T1 are
introduced. All theme/parity/contrast/component tests pass.

**Architecture:** A mechanical, repo-wide prefix rename guarded by a grep gate, plus a new brand
token block in `@cascade-ui/tokens`/`themes` (package names unchanged this tranche).

**Tech stack:** unchanged. Colors stay oklch.
