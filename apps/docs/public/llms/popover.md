# Popover

Anchored floating panel built on CSS Anchor Positioning + Popover API

## Install

```bash
npx cascade add popover
```

## Category

`overlay`

## States

- `open`
- `closed`

## Props

_No props._

## Design tokens

- `--cascivo-color-surface`
- `--cascivo-color-border`
- `--cascivo-radius-md`
- `--cascivo-shadow-md`
- `--cascivo-motion-enter`
- `--cascivo-motion-exit`

## Accessibility

- **WCAG level:** 2.2-AA
- **ARIA role:** `dialog`
- **Keyboard:** Escape

## Dependencies

- `@cascivo/core`

## Tags

overlay, floating, anchor, popover
