2. A "Responsive by default" docs page documents the scale + patterns and is linked in the nav; the docs
   app's off-scale literals are migrated onto the scale.
3. `breakpoint-allowlist.ts` is empty or only sanctioned/commented exceptions remain.
4. (If added) the optional `responsive` manifest field is non-required and drift-clean; otherwise
   explicitly skipped per Decision 10.
5. Full CLAUDE.md gate exits 0; the whole enrolled sweep is green; `ROADMAP-V20.md` DoD is fully checked
   and Status is ✅ Shipped.
