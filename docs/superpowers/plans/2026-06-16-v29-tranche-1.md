# v29 — Tranche 1: Route + Scaffold

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add `/blocks` and `/blocks/:name` routes to the landing SPA. Wire the nav link, SEO
head, and prerender entries. Define the `BlockMeta` type. Create skeleton `BlocksPage` and
`BlockDetailPage` components with layout shells (no real blocks yet — T2–T4 add those).

**Architecture:** Follows the exact pattern of every other non-home page: lazy imports in
`App.tsx`, entries in `ROUTES`, entries in `ROUTE_HEAD` and `PRERENDER_ROUTES` in
`route-head.ts`, and a nav link in `Header.tsx`. Both page components are thin skeletons that
will be filled out in T5.

**Tech Stack:** React 18, `@cascivo/core` (useSignals), CSS Grid + `@container`.
