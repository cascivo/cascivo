**Goal:** Retune the token system so cascade reads sharp, dense, and quiet: 6px controls,
monochrome primary, alpha hairline dark borders, rationed shadows, soft focus halo — and make
the radius knob real by migrating component CSS off primitive radius tokens.

**Architecture:** Almost everything is CSS-custom-property edits in `packages/tokens` and
`packages/themes`. One new semantic color pair (`--cascivo-color-primary[-fg|-hover|-active]`)
plus a Button CSS remap. The radius migration is a mechanical find/replace in component CSS with
an explicit mapping table.

**Tech stack:** CSS only (plus one Button CSS file). No TSX changes, no test logic changes.

| --------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
