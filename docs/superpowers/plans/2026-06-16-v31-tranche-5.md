# v31 — Tranche 5: Interaction Patterns (Swap, Chat Bubble, Radial Progress)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship three interaction-pattern components: `Swap` (animated toggle between two child states), `ChatBubble` (message UI primitive), and `RadialProgress` (circular conic-gradient progress). All three follow the standard four-file pattern.

**Architecture:** `Swap` uses a `useSignal` synchronized from the `checked` prop. `ChatBubble` is a pure display component — no signals. `RadialProgress` uses an inline `style` prop to set `--cascivo-radial-progress` (the only deliberate inline style in the cascade component set, required because `conic-gradient` cannot read inherited CSS custom properties).

**Tech Stack:** `useSignal`, `useSignals` from `@cascivo/core` for Swap. CSS `conic-gradient` for RadialProgress (Chrome 69+, Firefox 83+, Safari 12.1+). No new npm dependencies.

- [ ] **Step 1: Create `swap.tsx`**

  ```tsx
  import { useSignal, useSignals } from '@cascivo/core'
  import styles from './swap.module.css'

  export type SwapMode = 'rotate' | 'flip'

  export interface SwapProps {
    /** Content shown when checked */
    on: React.ReactNode
    /** Content shown when not checked */
    off: React.ReactNode
    checked?: boolean
    onChange?: (checked: boolean) => void
    mode?: SwapMode
    'aria-label'?: string
    className?: string
  }

  export function Swap({
    on,
    off,
    checked = false,
    onChange,
    mode = 'rotate',
    className,
    ...aria
  }: SwapProps) {
    useSignals()
    const isChecked = useSignal(checked)
    isChecked.value = checked

    function handleClick() {
      const next = !isChecked.value
      isChecked.value = next
      onChange?.(next)
    }

    return (
      <button
        type="button"
        role="switch"
        aria-checked={isChecked.value}
        className={[styles.swap, className].filter(Boolean).join(' ')}
        data-checked={isChecked.value || undefined}
        data-mode={mode}
        onClick={handleClick}
        {...aria}
      >
        <span className={styles.on} aria-hidden>
          {on}
        </span>
        <span className={styles.off} aria-hidden>
          {off}
        </span>
      </button>
    )
  }
  ```

  ```tsx
  import styles from './chat-bubble.module.css'

  export type ChatBubbleSide = 'start' | 'end'

  export interface ChatBubbleProps {
    children: React.ReactNode
    side?: ChatBubbleSide
    avatar?: React.ReactNode
    name?: string
    time?: string
    className?: string
  }

  export function ChatBubble({
    children,
    side = 'start',
    avatar,
    name,
    time,
    className,
  }: ChatBubbleProps) {
    return (
      <div
        className={[styles.bubble, className].filter(Boolean).join(' ')}
        data-side={side}
      >
        {avatar && (
          <div className={styles.avatar} aria-hidden>
            {avatar}
          </div>
        )}
        <div className={styles.content}>
          {(name || time) && (
            <div className={styles.meta}>
              {name && <span className={styles.name}>{name}</span>}
              {time && <time className={styles.time}>{time}</time>}
            </div>
          )}
          <div className={styles.body}>{children}</div>
        </div>
      </div>
    )
  }
  ```

- [ ] **Step 1: Create `radial-progress.tsx`**

  ```tsx
  import styles from './radial-progress.module.css'

  export type RadialProgressSize = 'sm' | 'md' | 'lg'
  export type RadialProgressVariant = 'primary' | 'info' | 'success' | 'warning' | 'error'

  export interface RadialProgressProps {
    /** 0–100 */
    value: number
    size?: RadialProgressSize
    variant?: RadialProgressVariant
    /** Center label. Defaults to `${value}%` */
    children?: React.ReactNode
    'aria-label'?: string
    className?: string
  }

  export function RadialProgress({
    value,
    size = 'md',
    variant = 'primary',
    children,
    className,
    ...aria
  }: RadialProgressProps) {
    const clamped = Math.min(100, Math.max(0, value))
    return (
      <div
        role="progressbar"
        aria-valuenow={clamped}
        aria-valuemin={0}
        aria-valuemax={100}
        className={[styles.radialProgress, className].filter(Boolean).join(' ')}
        data-size={size}
        data-variant={variant}
        style={{ '--cascivo-radial-progress': clamped } as React.CSSProperties}
        {...aria}
      >
        <span className={styles.label}>
          {children ?? `${clamped}%`}
        </span>
      </div>
    )
  }
  ```

- [ ] **Step 2: Create `radial-progress.module.css`**

  ```css
  @layer cascade.component {
    .radialProgress {
      --cascivo-radial-size: 5rem;
      --cascivo-radial-thickness: 0.375rem;
      --cascivo-radial-color: var(--cascivo-color-accent);

      display: inline-flex;
      align-items: center;
      justify-content: center;
      inline-size: var(--cascivo-radial-size);
      block-size: var(--cascivo-radial-size);
      border-radius: var(--cascivo-radius-full);

      /* Track */
      background-color: var(--cascivo-color-surface-2);

      /* Fill via conic-gradient — requires inline --cascivo-radial-progress */
      background-image: conic-gradient(
        var(--cascivo-radial-color) calc(var(--cascivo-radial-progress, 0) * 1%),
        transparent 0
      );

      /* Inner circle (donut effect) */
      box-shadow: inset 0 0 0 calc(var(--cascivo-radial-size) / 2 - var(--cascivo-radial-thickness))
        var(--cascivo-color-surface);
    }

    /* size variants */
    .radialProgress[data-size='sm'] { --cascivo-radial-size: 3.5rem; --cascivo-radial-thickness: 0.25rem; }
    .radialProgress[data-size='md'] { --cascivo-radial-size: 5rem;   --cascivo-radial-thickness: 0.375rem; }
    .radialProgress[data-size='lg'] { --cascivo-radial-size: 7rem;   --cascivo-radial-thickness: 0.5rem; }

    /* color variants */
    .radialProgress[data-variant='primary'] { --cascivo-radial-color: var(--cascivo-color-primary); }
    .radialProgress[data-variant='info']    { --cascivo-radial-color: var(--cascivo-color-info); }
    .radialProgress[data-variant='success'] { --cascivo-radial-color: var(--cascivo-color-success); }
    .radialProgress[data-variant='warning'] { --cascivo-radial-color: var(--cascivo-color-warning); }
    .radialProgress[data-variant='error']   { --cascivo-radial-color: var(--cascivo-color-error); }

    .label {
      font-size: 0.875em;
      font-weight: 600;
      color: var(--cascivo-color-text);
      line-height: 1;
    }
  }
  ```

  The `box-shadow: inset` technique creates the donut hole without a pseudo-element:
  a large inset shadow in the surface color hides the center of the conic-gradient.
  The thickness of the visible ring equals `size/2 - thickness`.
