**Goal:** A `MigrationGuide` section (anchor `#migrate`) on `/guides` that:

1. Maps the mental model: a "what transfers for free / what changes" two-column comparison.
2. Gives a numbered, copy-followable step-by-step path from a shadcn project to cascivo.
3. States honestly whether it's worth it, with the bundle/a11y/perf deltas sourced from **live bench
   data** and linked to `/performance` and `/accessibility`.
4. Ends in a clear next step (install command / QuickStart link).
5. Renders with zero horizontal overflow at 320–414px.

**Architecture:** New `apps/landing/src/pages/guides/MigrationGuide.tsx`, content in `guides/data.ts`,
styles in `landing.css`, claim 31 in `WhyCascadePage.tsx`. No package changes.

**Tech stack:** unchanged. Mobile-first CSS. `useSignals()` only if a signal is read.
