### Task 1: Signals-subscription grep gate

**Files:**

- Create: `scripts/checks/use-signals-gate.sh` (location: re-verify where existing check
  scripts live — `scripts/` has `factory/` and `registry/`; if an `audit:*` npm-script
  pattern exists in the root `package.json` (e.g. `audit:animation`), follow it)
- Modify: root `package.json` (add `audit:signals` script)

- [ ] **Step 1:** Write the gate — every React-app component file reading `.value` must
      call `useSignals()`:

```bash
#!/usr/bin/env bash
# Fails if a React-app component reads signal .value in render without useSignals().
# Heuristic: any .tsx under the React apps that contains `.value` AND `useSignal(`
# must also contain `useSignals()`. (Preact docs app excluded — natively reactive.)
set -euo pipefail
fail=0
for f in $(grep -rl --include='*.tsx' 'useSignal(' apps/landing/src apps/examples apps/bench 2>/dev/null); do
  if grep -q '\.value' "$f" && ! grep -q 'useSignals()' "$f"; then
    echo "MISSING useSignals(): $f"
    fail=1
  fi
done
exit $fail
```

- [ ] **Step 2:** Wire it: `"audit:signals": "bash scripts/checks/use-signals-gate.sh"` in
      the root package.json; run it:

```bash
pnpm run audit:signals
```

      Expected: exit 0. Sanity-check it guards: temporarily delete `useSignals()` from
      `FlagsRegion.tsx`, expect the file named + exit 1, restore.

- [ ] **Step 3:** Consider CI: if `.github/workflows/ci.yml`'s verify job runs other
      `audit:*` scripts (it runs `audit:animation` and `audit:bundle`), add
      `pnpm run audit:signals` beside them; mirror the step style.

- [ ] **Step 4: Commit**

```bash
git add scripts package.json .github/workflows/ci.yml
git commit -m "chore(v9): signals-subscription grep gate (audit:signals)"
```
