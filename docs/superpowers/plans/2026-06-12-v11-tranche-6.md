**Goal:** cascade tokens speak the W3C standard (export + import), the AI surface gains the
full-context file agents ask for plus a real integration guide, and WCAG/EAA conformance
becomes a generated, per-component, linkable page — pitch claims #5 and #6, and the
groundwork for a future Figma kit (deferred).

**Architecture:** A new `scripts/tokens/dtcg.ts` generator (export) + a `cascade tokens
import` CLI command (import → theme CSS, reusing the hoisted `generateThemeCss`).
llms-full + conformance page extend the existing generate pipeline in `scripts/llms/` —
same zero-drift model as everything else.

**Tech stack:** unchanged.
