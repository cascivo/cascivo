> re-read the tranche's "Current-state assumptions" before proceeding.**

**Goal:** Execute `docs/ROADMAP-V18.md` — publish the missing composition primitives in `@cascivo/core`,
run a formal parity audit against shadcn/ui + IBM Carbon, codify the gap as a machine-readable matrix,
queue every missing component into `factory-backlog.json`, drive the dark factory through them on the new
primitives, and close with a generated coverage page. Receipts, not adjectives.

**Architecture:** Two halves. (1) Primitives land in `packages/core/src/` exported from
`packages/core/src/index.ts`. (2) Parity audit lands in `docs/specs/parity-matrix.md` +
`scripts/parity/generate.ts` → `apps/docs/public/parity.json`, plus `factory-backlog.json` queue edits.
Components are generated by the factory into `packages/components/src/<name>/` and exported from
`packages/react/src/index.ts`. The coverage page lands in `apps/docs` (or `apps/landing`) reading
`parity.json`. Claims 34–36 on `apps/docs/src/pages/WhyCascadePage.tsx`.

**Tech stack:** unchanged. React 18+ + `@preact/signals-react`. No new runtime deps. Primitives are
signal-driven: `useSignal`/`useComputed`/`useSignalEffect`, never `useState`/`useEffect`/`useContext`.

## Key decisions

1. **Primitives before components.** T2–T4 land the substrate; T5–T6 build on it. Non-negotiable ordering.

2. **Factory builds components; tranches define specs + drive it.** T1 queues specs into
   `factory-backlog.json`; T5/T6 run the factory (`skills/` factory skill / `scripts/factory/`) in
   dependency order, then verify each against the component checklist (`CLAUDE.md` Part 3) + a11y.

3. **Parity matrix is generated + drift-gated.** `docs/specs/parity-matrix.md` (human source) →
   `parity.json` (generated). Hand-editing the JSON is forbidden; regen + `git diff --exit-code` enforces.

4. **Honest coverage.** covered / partial / gap / by-convention — each competitor entry maps to a real
   cascivo component or an explicit note. No inflation; partials carry a reason.

5. **No new runtime deps.** `Slot` uses `cloneElement` + existing `composeRefs`/`mergeProps`. Anchor
   positioning = CSS anchor positioning + JS fallback (progressive, per v13 stance). `useMediaQuery`
   wraps `matchMedia` in `useSignalEffect`.

6. **Signal rules everywhere.** No `useState`/`useEffect`/`useContext`/`useReducer` in any new core code.
   `useSignalEffect` for DOM side effects; `useControllableSignal` for controlled props; `useRef` for DOM
   refs only.

7. **Reconcile, don't duplicate, the backlog.** T1 upgrades the 13 relevant `pending` entries
   (`icon-button`, `notification`, `structured-list`, `tile`, `scroll-area`, `calendar`,
   `date-range-picker`, `tree-view`, `color-picker`, `carousel`, `timeline`, `data-list`, `collapsible`,
   `code-block`→`code-snippet`, `splitter`→`resizable`) and only _adds_ the genuinely-missing ones.

8. **Claims 34–36 land in T7** on `WhyCascadePage.tsx`, each with a receipt linking to the generated
   coverage page. Commit per task; full gate runs at T7.

9. **Coverage page can't overstate.** Renders from `parity.json`; a component shows as covered only when
   built + exported from `@cascivo/react`.
