# v30 — Tranche 2: Below-Fold Code Splitting

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Convert all below-fold home sections to `React.lazy()` imports. Currently only `ChartShowcase` and `RelayConsole` are lazy; twelve other sections (including heavy interactive ones like `SignalsDemo`, `AgentLayer`, and `ThemeDemo`) are parsed and evaluated before React renders the first pixel. After this tranche each below-fold section is a separate JS chunk downloaded after the above-the-fold content paints.

**Architecture:** Pure `App.tsx` change. Use `React.lazy()` + `<Suspense fallback={<SectionFallback />}>` for every below-fold section. The existing `SectionFallback` component and `lazy-fallback` CSS class are already in place — they just need to be applied more broadly. Add a `height` prop to `SectionFallback` for sections that need a specific reserved height.

**Tech Stack:** React `lazy()`, `Suspense`, approximate section heights from dev tools.

### Task 2: Extend `SectionFallback` to accept a `height` prop

**Files:**

- Modify: `apps/landing/src/App.tsx`

- [ ] **Step 1: Update `SectionFallback`**

  The current component is:

  ```tsx
  function SectionFallback({ tall = false }: { tall?: boolean }) {
    return (
      <div className={tall ? 'lazy-fallback lazy-fallback--tall' : 'lazy-fallback'} aria-hidden />
    )
  }
  ```

  Replace with:

  ```tsx
  function SectionFallback({ tall = false, height }: { tall?: boolean; height?: number }) {
    return (
      <div
        className={tall ? 'lazy-fallback lazy-fallback--tall' : 'lazy-fallback'}
        style={height !== undefined ? { minBlockSize: height } : undefined}
        aria-hidden
      />
    )
  }
  ```

  The `height` prop accepts a pixel value for sections with a known approximate height.
  Sections without a `height` prop fall back to the existing `tall` / default CSS behaviour.
