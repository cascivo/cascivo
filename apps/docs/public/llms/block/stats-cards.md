# StatsCards

Grid of KPI stat cards with trend badges.

## Install

```bash
npx cascade add block/stats-cards
```

## Category

`display`

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `stats` | `Stat[]` | no | — | KPI stat data |

## Examples

### Default

Demo KPI stats

```tsx
<StatsCards />
```

## Accessibility

- **WCAG level:** AA
- **ARIA role:** `generic`

## Dependencies

- `@cascade-ui/react`

## Tags

block, stats, kpi, cards
