# ConsoleApp

Carbon-parity console shell: ShellHeader + icon-rail SideNav + HeaderPanel notifications/switcher + collapsible aside + main content.

## Install

```bash
npx cascade add block/console-app
```

## Category

`display`

## Props

_No props._

## Examples

### Default

Full console shell demo

```tsx
<ConsoleApp />
```
