# v33 — Tranche 1: Home Re-render Demo — One Form, Two Live Counters

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rework the "Count the re-renders" section (`#signals`) so a single visible form drives **two render-count badges** shown together, and a `prefers-reduced-motion`-aware typewriter auto-types into the field so the counters tick live on load — signals stays near zero, useState climbs per keystroke.

**Architecture:** The visible form binds to module-level signals (the "signals" twin; its form component never re-renders). A mounted `StateTwin` mirrors the same signals into `useState` via `useSignalEffect`, re-rendering once per keystroke (the sanctioned `useState` exception). A typewriter `useSignalEffect` appends characters to the signals. No `useEffect`.

**Tech Stack:** React + `@cascivo/core` signals, `@cascivo/components` (Input/Checkbox/Button/Badge/Card). CSS in `landing.css`.

- [ ] **Step 3: Replace `StateForm` with a minimal `StateTwin`.** The twin is no longer a second visible form — it is the useState re-render counter. It mirrors the shared signals into local `useState` and re-renders once per keystroke:

  ```tsx
  function StateTwin() {
    // Sanctioned useState exception: this twin is the comparison subject.
    // eslint-disable-next-line react-hooks/rules-of-hooks
    const [, setName] = useState('')
    // eslint-disable-next-line react-hooks/rules-of-hooks
    const [, setEmail] = useState('')
    // eslint-disable-next-line react-hooks/rules-of-hooks
    const [, setNewsletter] = useState(false)
    const renders = useRef(0)
    const mounted = useRef(false)

    // Each signal write re-runs this effect → setState → full re-render of StateTwin.
    useSignalEffect(() => {
      setName(sigName.value)
      setEmail(sigEmail.value)
      setNewsletter(sigNewsletter.value)
    })

    if (mounted.current) {
      renders.current++
      queueMicrotask(() => {
        stateFormRenders.value = renders.current
      })
    } else {
      mounted.current = true
    }
    return null // counts only; the visible form is the signals twin
  }
  ```

  Import `useSignalEffect` from `@cascivo/core`. Keep `useRef`/`useState` from `react`.
