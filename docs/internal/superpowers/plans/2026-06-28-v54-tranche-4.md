# v54 — Tranche 4: Keyboard Navigation + Undoable Insertion

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the open menu operable from the keyboard and apply the chosen command undoably (S-4/S-5/S-6). Lens:
**CodeMirror/Monaco.** Both the mechanism and the precedent already exist: the editor **adds bindings conditionally
while an overlay is open** — `Escape` only `if (findOpen.value)` (`code-editor.tsx:507-512`) — and `applyEdit` already
performs an undoable replace (`code-editor.tsx:457-463`). T4 reuses both: nav keys bound only while `slashOpen`, and
selection routed through `applyEdit`/`run`.

**Architecture:** a conditional keymap block + a `selectCommand` + the combobox a11y wiring.
- **Keymap-while-open**: inside `handleKeyDown` (`code-editor.tsx:474`), add an `if (slashOpen.value)` block (sibling to
  the `if (findOpen.value)` one) binding `ArrowDown`/`ArrowUp` (move `slashIndex`, clamped to the filtered length),
  `Enter` and `Tab` (select the active command), and `Escape` (close + keep the literal `/query` in the text). Each
  returns `true` so `dispatch` `preventDefault`s — these keys must not also move the textarea caret or insert a tab
  while the menu is up.
- **`selectCommand(index)`**: resolve the active `SlashCommand`; replace the trigger span `[slashStart.value, caret)`
  (the `/` + query) — for an `insert` command via `applyEdit({ from: slashStart.value, to: caret }, cmd.insert)`; for a
  `run` command, `applyEdit(..., '')` to remove the trigger then `cmd.run(handle)`. Close the menu; return focus to the
  textarea. One undoable step.
- **Combobox a11y**: set `aria-activedescendant` on the `<textarea>` to the active option's id while `slashOpen`, and
  `aria-expanded`/`aria-controls` referencing the listbox — the ARIA combobox-over-textbox pattern, without disturbing
  native IME/undo.

**Tech Stack:** `code-editor.tsx` only (keymap block + `selectCommand` + ARIA attrs); reuses `applyEdit`/`commit`/`history`.
No new dependency.
