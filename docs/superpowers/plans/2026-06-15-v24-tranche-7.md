# v24 — Tranche 7: Track Settings Section

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the Settings section for Cascade Track, matching `linear1.png` (Preferences) and `linear2.png` (Workspace). Two sub-pages selectable via a `Tabs` bar. Preferences: General settings (`Input` for display name, `Toggle` for notifications), Interface & theme (`Select` for font size, `Toggle` for pointer cursors, `Radio` group for Light / Dark that updates the persisted AppShell theme signal). Workspace: logo `FileUploader`, name + URL `Input`s, fiscal-month + region `Select`s, danger zone — "Delete workspace" destructive `Button` inside an `AlertDialog`.

**Architecture:** `Settings.tsx` wraps Preferences and Workspace sub-pages via `Tabs`. Both sub-pages are self-contained TSX files. The Light/Dark `Radio` reads and writes the AppShell's `theme` signal. The `AlertDialog` fires a no-op callback (workspace deletion is in-memory only). Rendered when `currentSection.value === 'settings'` in `App.tsx`.

**Tech Stack:** `@cascivo/react` (Tabs, TabsList, TabsTrigger, TabsContent, Input, Toggle, Select, Radio, RadioGroup, FileUploader, AlertDialog, AlertDialogTrigger, AlertDialogContent, AlertDialogTitle, AlertDialogDescription, AlertDialogAction, AlertDialogCancel, Button), `@cascivo/storage` (persistedSignal), `@cascivo/i18n`.

- [ ] **Step 3: Create Preferences.tsx**

  The theme radio must write to the AppShell's shared persisted theme signal. That signal is
  exported from `@cascivo/example-kit` at `kit.appShell.theme` storage key. To avoid coupling
  to kit internals, expose a `setTheme(t: 'dark' | 'light')` helper from the kit's
  `app-shell.tsx` by adding an export there.

  In `apps/examples/kit/src/app-shell.tsx`, add after the `theme` signal declaration:
  ```ts
  export function setAppTheme(t: AppShellTheme) { theme.value = t }
  export function getAppTheme() { return theme.value }
  ```

  Then create `apps/examples/track/src/sections/settings/Preferences.tsx`:

  ```tsx
  'use client'
  import { useSignal, useSignals } from '@cascivo/core'
  import { t } from '@cascivo/i18n'
  import { Input, Toggle, Select, RadioGroup, Radio } from '@cascivo/react'
  import { setAppTheme, getAppTheme } from '@cascivo/example-kit'
  import { msg } from '../../i18n'
  import styles from './Preferences.module.css'

  const FONT_SIZE_OPTIONS = [
    { value: 'default', label: t(msg.prefFontDefault) },
    { value: 'large', label: t(msg.prefFontLarge) },
  ]

  export function Preferences() {
    useSignals()

    const displayName = useSignal('Adam Urban')
    const notificationsEnabled = useSignal(true)
    const pointerCursors = useSignal(false)
    const currentTheme = useSignal<'dark' | 'light'>(getAppTheme() as 'dark' | 'light')

    return (
      <div className={styles['root']}>
        {/* General section */}
        <section className={styles['section']}>
          <h2 className={styles['sectionTitle']}>{t(msg.prefSectionGeneral)}</h2>

          <div className={styles['row']}>
            <div className={styles['rowLabel']}>
              <label htmlFor="display-name" className={styles['label']}>{t(msg.prefDisplayName)}</label>
              <p className={styles['desc']}>{t(msg.prefDisplayNameDesc)}</p>
            </div>
            <Input
              id="display-name"
              value={displayName.value}
              onChange={(e) => { displayName.value = e.target.value }}
            />
          </div>

          <div className={styles['row']}>
            <div className={styles['rowLabel']}>
              <span className={styles['label']}>{t(msg.prefNotifications)}</span>
              <p className={styles['desc']}>{t(msg.prefNotificationsDesc)}</p>
            </div>
            <Toggle
              checked={notificationsEnabled.value}
              onCheckedChange={(v) => { notificationsEnabled.value = v }}
              aria-label={t(msg.prefNotifications)}
            />
          </div>
        </section>

        {/* Interface and theme section */}
        <section className={styles['section']}>
          <h2 className={styles['sectionTitle']}>{t(msg.prefSectionTheme)}</h2>

          <div className={styles['row']}>
            <div className={styles['rowLabel']}>
              <span className={styles['label']}>{t(msg.prefFontSize)}</span>
              <p className={styles['desc']}>{t(msg.prefFontSizeDesc)}</p>
            </div>
            <Select
              options={FONT_SIZE_OPTIONS}
              value="default"
              onChange={() => {}}
              aria-label={t(msg.prefFontSize)}
            />
          </div>

          <div className={styles['row']}>
            <div className={styles['rowLabel']}>
              <span className={styles['label']}>{t(msg.prefPointerCursors)}</span>
              <p className={styles['desc']}>{t(msg.prefPointerCursorsDesc)}</p>
            </div>
            <Toggle
              checked={pointerCursors.value}
              onCheckedChange={(v) => { pointerCursors.value = v }}
              aria-label={t(msg.prefPointerCursors)}
            />
          </div>

          <div className={styles['row']}>
            <div className={styles['rowLabel']}>
              <span className={styles['label']}>{t(msg.prefThemeLabel)}</span>
              <p className={styles['desc']}>{t(msg.prefThemeDesc)}</p>
            </div>
            <RadioGroup
              name="app-theme"
              value={currentTheme.value}
              onChange={(v) => {
                currentTheme.value = v as 'dark' | 'light'
                setAppTheme(v as 'dark' | 'light')
              }}
            >
              <Radio value="light" label={t(msg.prefThemeLight)} />
              <Radio value="dark" label={t(msg.prefThemeDark)} />
            </RadioGroup>
          </div>
        </section>
      </div>
    )
  }
  ```
