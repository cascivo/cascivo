**Goal:** Ship two new first-party themes (`flat.css` and `minimal.css`) as separate CSS files in
`packages/themes/`. Update docs/Storybook theme switcher to 4 options. Add both as archetypes in
the `cascade:create-theme` skill. Update landing page to showcase all 4 themes.

**Architecture:** Both themes override the semantic layer only (same rule as light/dark/warm). They
use the oklch token foundation established in T4. Both must pass the contrast gate with the
documented distinction that decorative borders (`--cascade-border-subtle`) are exempt from the 3:1
graphics requirement. The themes are additive — no existing theme file is modified.

**Tech stack:** CSS (oklch), TypeScript (theme switcher, skill), Playwright (visual regression
snapshots for two new themes).

**Conventions:** Theme files override only semantic tokens in `[data-theme="flat"]` and
`[data-theme="minimal"]` selectors. No component-level token overrides. Run `vp check` before
each commit.

### Task 1: Create `flat.css`

**Files:**
- Create: `packages/themes/src/flat.css`

- [ ] **Step 1: Define the flat theme**

Flat design: zero elevation, high-contrast borders, solid fills, square corners, confident accent.

```css
/**
 * cascade flat theme — zero shadows, zero radius, high-contrast borders.
 * Depth is conveyed by border contrast only. Accent: vivid green oklch(0.5 0.2 145).
 */
[data-theme="flat"] {
  /* Backgrounds */
  --cascade-color-background: oklch(1 0 0);
  --cascade-color-surface:    oklch(0.970 0 0);
  --cascade-color-surface-2:  oklch(0.940 0 0);

  /* Foreground */
  --cascade-color-foreground:       oklch(0.100 0 0);
  --cascade-color-foreground-muted: oklch(0.400 0 0);

  /* Accent: confident green */
  --cascade-color-accent:            oklch(0.500 0.200 145);
  --cascade-color-accent-foreground: oklch(1 0 0);
  --cascade-color-accent-hover:      oklch(0.440 0.200 145);

  /* Borders: 2px solid high-contrast — depth via border only */
  --cascade-border-subtle:  oklch(0.850 0 0);
  --cascade-border-default: oklch(0.600 0 0);
  --cascade-border-strong:  oklch(0.200 0 0);

  /* Zero radius — flat means flat */
  --cascade-radius-base: 0;

  /* Zero shadows — borders provide depth */
  --cascade-shadow-xs: none;
  --cascade-shadow-sm: none;
  --cascade-shadow-md: none;
  --cascade-shadow-lg: none;

  /* Focus: solid 2px outline (visible without radius) */
  --cascade-ring-width: 2px;
  --cascade-ring-offset: 2px;
  --cascade-ring-color: oklch(0.100 0 0);

  /* Semantic */
  --cascade-color-error:   oklch(0.550 0.200 22);
  --cascade-color-warning: oklch(0.600 0.180 50);
  --cascade-color-success: oklch(0.500 0.200 145);

  /* Density — tighter line-height for flat feel */
  --cascade-control-height-sm: 1.625rem;
  --cascade-control-height-md: 2.125rem;
  --cascade-control-height-lg: 2.625rem;
}
```

- [ ] **Step 2: Verify zero-radius focus ring visibility**

Flat theme has `--cascade-radius-base: 0` which means no border-radius on focus ring either.
The `--cascade-ring-color: oklch(0.100 0 0)` (near-black) ensures the solid outline is always
visible. This is the documented resolution for edge case 6.

- [ ] **Step 3: Run contrast check**

```bash
node scripts/quality/contrast-check.ts flat
```

Expected: foreground/background ≥4.5:1; accent/accent-foreground ≥4.5:1; border-default/background ≥3:1.

- [ ] **Step 4: Commit**

```bash
git add packages/themes/src/flat.css
git commit -m "feat(themes): flat.css — zero shadows, zero radius, 2px borders, vivid green accent"
```

### Task 2: Create `minimal.css`

**Files:**
- Create: `packages/themes/src/minimal.css`

- [ ] **Step 1: Define the minimal theme**

Minimal: monochrome, generous spacing, near-invisible decorative borders, single barely-there
popover shadow, hierarchy via typography not boxes.

```css
/**
 * cascade minimal theme — monochrome, generous density, barely-there borders.
 * Hierarchy via type weight and size. Interactive states via lightness shift only.
 * Decorative borders (--cascade-border-subtle) are intentionally below 3:1 — they
 * are non-informational dividers; all state-bearing borders remain high-contrast.
 */
[data-theme="minimal"] {
  /* Backgrounds: warm off-white */
  --cascade-color-background: oklch(0.980 0.005 80);
  --cascade-color-surface:    oklch(0.970 0.006 80);
  --cascade-color-surface-2:  oklch(0.955 0.007 80);

  /* Foreground: near-black, slightly warm */
  --cascade-color-foreground:       oklch(0.150 0.010 50);
  --cascade-color-foreground-muted: oklch(0.500 0.015 60);

  /* Accent: subdued link-blue (chroma < 0.06) — no vivid accent color */
  --cascade-color-accent:            oklch(0.500 0.060 260);
  --cascade-color-accent-foreground: oklch(1 0 0);
  --cascade-color-accent-hover:      oklch(0.430 0.060 260);

  /* Borders: near-invisible decorative; high-contrast for state borders */
  --cascade-border-subtle:  oklch(0.920 0.005 80);  /* decorative — below 3:1 by design */
  --cascade-border-default: oklch(0.850 0.008 80);  /* standard — still low-contrast */
  --cascade-border-strong:  oklch(0.350 0.020 50);  /* state/emphasis — always high-contrast */

  /* Rounder radius — generous feel */
  --cascade-radius-base: 0.750rem;

  /* Single barely-there shadow for floating surfaces only */
  --cascade-shadow-xs: none;
  --cascade-shadow-sm: none;
  --cascade-shadow-md: 0 1px 8px oklch(0 0 0 / 0.05);
  --cascade-shadow-lg: 0 2px 16px oklch(0 0 0 / 0.08);

  /* Focus: thin 1.5px foreground-color ring at 40% */
  --cascade-ring-width: 1.5px;
  --cascade-ring-offset: 3px;
  --cascade-ring-color: oklch(0.150 0.010 50 / 0.4);

  /* Density: generous — 40-44px controls */
  --cascade-control-height-sm: 2.000rem;   /* 32px */
  --cascade-control-height-md: 2.500rem;   /* 40px */
  --cascade-control-height-lg: 2.875rem;   /* 46px */

  /* Semantic */
  --cascade-color-error:   oklch(0.560 0.160 22);
  --cascade-color-warning: oklch(0.650 0.140 60);
  --cascade-color-success: oklch(0.560 0.140 145);
}
```

- [ ] **Step 3: Run contrast check (excluding border-subtle)**
