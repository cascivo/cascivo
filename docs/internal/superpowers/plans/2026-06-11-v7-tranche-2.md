**Goal:** Replace ComponentGrid with "Relay" — a dense, realistic deploy-ops console composed
from ≥25 distinct cascade components with fixture data that reads like a shipped product, one
permanently warm-themed card proving scoped theming, and working interactions (sort, modal,
toast, switches, tabs).

**Architecture:** `src/demo/` module: `data.ts` (all fixture data, typed, zero randomness),
`RelayConsole.tsx` (frame + layout grid), one file per region. The console is theme-driven by
the page (`<html data-theme>`); only the on-call card pins `data-theme="warm"`.

**Tech stack:** `@cascivo/react`, `@cascivo/charts`, tokens-only CSS.
