### Task 5: Full CLAUDE.md gate

- [ ] **Step 1: Run all six gate commands**

  ```sh
  pnpm exec vp check
  ```
  Expected: exits 0 (fmt + lint + tsc).

  ```sh

  ```
  Expected: exits 0 (all packages build).

  ```sh

  ```
  Expected: exits 0 (type-check all packages).

  ```sh

  ```
  Expected: exits 0 (all unit + Playwright tests pass, including coverage.test.ts).

  ```sh
  pnpm regen
  pnpm exec vp check --fix
  git diff --exit-code
  ```
  Expected: exits 0 (regenerated artifacts committed, no drift).

  ```sh
  pnpm breakpoint:check
  ```
  Expected: exits 0 (no off-scale breakpoint literals).

- [ ] **Step 2: If any gate fails, fix before proceeding**

  Do not commit with a failing gate. Diagnose the specific failure:
  - `vp check` failure → fix lint/format/type error in the flagged file
  - `pnpm build` failure → fix the import or missing export
  - `pnpm test` failure → fix the broken test or the implementation
  - `git diff --exit-code` → run `pnpm regen` again and commit the generated artifacts
  - `breakpoint:check` → replace any non-canonical `@media`/`@container` width literals
