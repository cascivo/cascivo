# v46 — Tranche 5: Per-Instance Theming + Active-Line Gutter + Bracket Matching

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close the remaining 🟠 affordance gaps. (1) **Per-instance theming with live switching** (gap-analysis
**#6**): a `theme` prop of `--cascivo-editor-*` overrides spread onto the root, swappable at runtime — the
"Zen-mode" Compartment behavior, without CodeMirror's Compartment machinery. (2) **Active-line gutter highlight**
and **bracket matching** (gap-analysis **#8**): style the active gutter line off the caret-line property already
tracked, and decorate the matching bracket pair adjacent to the caret via the T4 decoration seam. All opt-in and
additive.

**Architecture:** Theming — the root is signal-rendered with an inline `style` (already used for
`--cascivo-editor-tab-size`). Add `theme?: EditorTheme` (a `Partial<Record<EditorCustomProp, string>>`) and spread
it into `rootStyle`; because render is reactive, replacing the object re-themes live. Global `data-theme` stays
the default; the prop only overrides the editor-scoped custom properties. Active-line gutter — `code-editor.tsx`
already sets `--cascivo-editor-caret-line` in the caret listener; add a gutter style that highlights the line at
that index (a CSS marker positioned like `.currentLine`, or a class toggled per gutter row). Prefer CSS-first: a
single positioned element in the gutter keyed off the same custom property (mirrors `.currentLine`). Bracket
matching — `brackets.ts` finds the partner of a bracket adjacent to the caret (scan with a depth counter,
respecting only literal brackets; do not parse strings/comments for v1) and emits two `Decoration`s through the
T4 seam, behind a `bracketMatching` prop.

**Tech Stack:** owned TypeScript; `@cascivo/core` signals; CSS module (static fallback before any progressive
declaration; `fallback:check`); Vitest. No banned hooks.
