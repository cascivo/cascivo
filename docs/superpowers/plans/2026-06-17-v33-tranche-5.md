### Task 1: Add the percentage-delta helper

**Files:** Modify `apps/landing/src/pages/perf-data.ts`

- [ ] **Step 1: Add a `pctDelta` helper** that returns a formatted signed delta string or `null` when not meaningful:

  ```ts
  /** Signed % difference of `val` vs cascivo `base`. null when not meaningful. */
  export function pctDelta(val: number | undefined, base: number | undefined): string | null {
    if (val === undefined || base === undefined || base === 0) return null
    const pct = Math.round(((val - base) / base) * 100)
    if (pct === 0) return null
    return `${pct > 0 ? '+' : '−'}${Math.abs(pct)}%`
  }
  ```

  Use the Unicode minus (`−`, U+2212) for negatives to match typographic conventions; verify it passes lint/format.
