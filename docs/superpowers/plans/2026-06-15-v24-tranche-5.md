# v24 — Tranche 5: Track Sidebar + Sub-tabs

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Restructure the Track (Linear) sidebar from two flat items (Board, List) into Linear's three-tier grouped nav matching `linear0.png`. Add a notification count `Badge` to Notifications. Add sub-tabs (All Issues / Active / Backlog) to the issues list content header. Add a Settings nav item as a stub (implemented in T7).

**Architecture:** The Track `App.tsx` migrates from `navItems` to `navGroups`. A new `issueTab` signal (All / Active / Backlog) is added to the app-level signals; it drives issue list filtering. The `Board` view is unaffected. A `Settings` section signal value is added to `Section` type.

**Tech Stack:** `@cascivo/react` (Tabs, TabsList, TabsTrigger, TabsContent, Badge), `@cascivo/example-kit` (AppShell with navGroups), `@cascivo/i18n`, `@cascivo/core` (signal).

  ```tsx

  const notifBadge = (

  )

  const navGroups: SideNavGroup[] = [
    {
      // Personal — no section header (label omitted)

          active: false,
          onClick: (e) => {
            e.preventDefault()
          },
        },
        {
          label: t(msg.navMyIssues),
          active: false,
          onClick: (e) => {
            e.preventDefault()
          },
        },
        {
          label: t(msg.navNotifications),
          active: false,
          icon: notifBadge,
          onClick: (e) => {
            e.preventDefault()
          },
        },
      ],
    },
    {
      label: t(msg.navGroupFavorites),
      items: [], // empty — placeholder text rendered via CSS ::after or EmptyState
    },
    {
      label: t(msg.navGroupTeams),
      items: [
        {
          label: t(msg.navTeamName),
          active: currentSection.value === 'issues',
          items: [
            {
              label: t(msg.navSubIssues),
              href: '#',
              active: currentSection.value === 'issues',
            },
            {
              label: t(msg.navSubCycles),
              href: '#',
              active: false,
            },
            {
              label: t(msg.navSubProjects),
              href: '#',
              active: false,
            },
            {
  ```
