# v29 — Tranche 2: Auth Blocks

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement `auth-login` and `auth-signup` blocks with full TSX, CSS module, meta, and
test files. Both blocks use only `@cascivo/react` components. Both pass lint, type-check, and
the component test suite.

**Architecture:** Each block is a self-contained directory under
`packages/components/src/blocks/<name>/`. Blocks use `useSignal` for local form state (controlled
inputs) and `useSignalEffect` for any side effects. No `useState`, `useEffect`, `useContext`.
CSS uses only `var(--cascivo-*)` tokens — no hardcoded colours or sizes. Layout uses CSS Grid
with `@container` for responsive behaviour.

**Tech Stack:** `@cascivo/core` (useSignal, useSignals), `@cascivo/react` (Button, Input,
Checkbox, Card), Vitest + @testing-library/react.

- [ ] **Step 2: Create the block component**

  Create `packages/components/src/blocks/auth-login/auth-login.tsx`:

  ```tsx
  import { useSignal, useSignals } from '@cascivo/core'
  import { Button } from '@cascivo/react/button'
  import { Card } from '@cascivo/react/card'
  import { Checkbox } from '@cascivo/react/checkbox'
  import { Input } from '@cascivo/react/input'
  import styles from './auth-login.module.css'

  export function AuthLogin() {
    useSignals()
    const email = useSignal('')
    const password = useSignal('')
    const remember = useSignal(false)

    return (
      <div className={styles.root}>
        <div className={styles.card}>
          <Card>
            <h1 className={styles.heading}>Sign in</h1>
            <p className={styles.subheading}>Enter your credentials to continue</p>
            <form className={styles.form} onSubmit={(e) => e.preventDefault()}>
              <div className={styles.field}>
                <label htmlFor="login-email" className={styles.label}>
                  Email
                </label>
                <Input
                  id="login-email"
                  type="email"
                  placeholder="you@example.com"
                  autoComplete="email"
                  value={email.value}
                  onChange={(e) => {
                    email.value = e.target.value
                  }}
                />
              </div>

              <div className={styles.field}>
                <div className={styles.passwordRow}>
                  <label htmlFor="login-password" className={styles.label}>
                    Password
                  </label>
                  <a href="/forgot-password" className={styles.forgotLink}>
                    Forgot password?
                  </a>
                </div>
                <Input
                  id="login-password"
                  type="password"
                  placeholder="••••••••"
                  autoComplete="current-password"
                  value={password.value}
                  onChange={(e) => {
                    password.value = e.target.value
                  }}
                />
              </div>

              <div className={styles.rememberRow}>
                <Checkbox
                  id="login-remember"
                  checked={remember.value}
                  onCheckedChange={(v) => {
                    remember.value = Boolean(v)
                  }}
                />
                <label htmlFor="login-remember">Remember me</label>
              </div>

              <Button type="submit" variant="primary" style={{ inlineSize: '100%' }}>
                Sign in
              </Button>
            </form>

            <p className={styles.footer}>
              Don&apos;t have an account?{' '}
              <a href="/signup" className={styles.footerLink}>
                Create an account
              </a>
            </p>
          </Card>
        </div>
      </div>
    )
  }
  ```

  > **Note on imports:** The exact `@cascivo/react` import paths depend on how the package
  > exports are configured. Read `packages/react/package.json` and use the correct paths.
  > If the package exports everything from the root (`import { Button } from '@cascivo/react'`),
  > use that form. Match what other blocks and tests in the repo already use.
