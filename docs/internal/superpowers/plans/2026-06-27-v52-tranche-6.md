# v52 — Tranche 6: High-Demand Exotic Types — Sankey, Sunburst, Stream, Calendar

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the four highest-demand chart types nivo has and cascivo lacks (N-5): **Sankey** (flow), **Sunburst**
(radial hierarchy), **Stream** (wiggle/silhouette stacked area), and **Calendar** (GitHub-style day-grid heatmap).
This closes the v51 Sankey/Sunburst deferral and adds the two most-requested density types. Explicitly **out of
scope** (Decision 5): Chord, Bump, Swarm, Waffle, Marimekko, Icicle, ParallelCoordinates, CirclePacking, PolarBar,
TimeRange, Wordcloud, ViolinPlot, GeoMap/Choropleth, and Network (covered by `@cascivo/flow`). Lens: **nivo**.

**Architecture:** each type needs a from-scratch, dependency-free layout in `engine/*` (Decision 4), then a component
following the established chart contract (`ChartFrame`, `role="img"`, fallback table, optional tooltip, theme colors,
i18n, a sibling `*.meta.ts` with `intent`, tests, exports, generated docs).
- **Sankey** (`engine/sankey.ts`): assign nodes to ranks (longest-path), order within a rank to reduce crossings,
  size node heights by throughput, route links as cubic ribbons. Reuse `arcPath`-style cubic emission.
- **Sunburst** (`engine/hierarchy.ts` partition): a radial partition of a tree — each node an annular segment via the
  existing `arcPath`; depth → radius ring, value → angular sweep. (`treemap.ts`'s squarified layout is the rectilinear
  cousin; this is its radial sibling.)
- **Stream** (`engine/stream.ts`): extend `stackSeries` with `wiggle`/`silhouette` offset so the stack centers on a
  flowing baseline; render with the existing `areaPath` + a curve.
- **Calendar** (`charts/calendar`): a day grid (weeks × weekdays) of value-colored cells over a date range — the
  Heatmap's date-native cousin; reuse the heatmap cell + the CVD-safe sequential ramp.

**Tech Stack:** SVG + the existing `arcPath`/`areaPath`/`stackSeries` + new layouts; `ChartFrame`; `@cascivo/i18n`;
`--cascivo-chart-*` tokens. No new dependency.
