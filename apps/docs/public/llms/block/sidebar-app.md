# SidebarApp

Full app shell with collapsible side navigation and top header.

## Install

```bash
npx cascade add block/sidebar-app
```

## Category

`display`

## Props

_No props._

## Examples

### Default

App with sidebar navigation

```tsx
<SidebarApp />
```

## Accessibility

- **WCAG level:** AA
- **ARIA role:** `generic`

## Dependencies

- `@cascade-ui/react`
- `layout/app-shell`

## Tags

block, sidebar, navigation, app-shell
