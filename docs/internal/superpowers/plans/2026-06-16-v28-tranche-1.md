# v28 — Tranche 1: Route + Page Scaffold

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the `/create` route to the landing SPA. Wire the nav link, SEO head, and prerender
entry. Create the `CreatePage` component with the three-panel CSS Grid layout (empty panels,
no interactivity yet — that comes in T2–T5).

**Architecture:** Follows the exact pattern of every other non-home page: lazy import in `App.tsx`,
entry in `ROUTES` map, entry in `ROUTE_HEAD` and `PRERENDER_ROUTES` in `route-head.ts`, and a nav
link in `Header.tsx`. `CreatePage.tsx` renders the three-panel shell with placeholder content.

**Tech Stack:** React 18, `@cascivo/core` (useSignals), CSS Grid + `@container`.
