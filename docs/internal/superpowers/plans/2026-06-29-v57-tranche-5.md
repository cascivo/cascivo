# v57 — Tranche 5: Correct the Record & Close the Loop — Layout/Spacing Discoverability + Wrap-up

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix the *real* residual of the report's "no layout primitives / no spacing utilities → inline-style drift"
complaint, which the analysis showed is **inaccurate** (nine primitives + a documented `--cascivo-space-*` scale
already ship) — the genuine gap is **discoverability**: they install as `layout/*` (not bare `stack`) and are named
`Stack`/`Columns`/`AutoGrid` (not `Flex`/`Box`/`HStack`). Make them findable from the obvious queries and write the
guide that turns inline styles into primitives + tokens — **without** shipping utility classes (an anti-goal). Then
close the loop: append the maintainer resolution log to the feedback file and flip the roadmap to Shipped (M-5,
C-1/C-2; Master Decisions 6/7).

**Architecture:** small CLI ergonomics + one cookbook + the wrap-up edits. **No new component, no utility CSS.**
- **CLI** (`packages/cli/src/commands/add.ts` + `list.ts`): resolve a bare name to its namespaced entry when
  unambiguous (`stack` → `layout/stack`), and map common aliases (`flex`/`hstack`/`vstack`/`box` → `stack`;
  `gap`/`spacer` → `spacer`) — reusing the synonym map defined in T3. `cascivo list` surfaces the `layout/*` group
  prominently with one-line descriptions.
- **`docs/cookbooks/layout-and-spacing.md`**: a before/after guide — the report's `style={{ display:'flex', gap:16 }}`
  drift replaced with `<Stack gap={4}>`, `<Grid>`, `<AutoGrid>`, `<Columns>`, `<Center>`, `<Section>`, and the
  sanctioned escape hatch `var(--cascivo-space-N)` for one-off custom CSS. Names the `Flex/Box/HStack`→primitive
  mapping explicitly so the next reader's mental model maps on.

**Tech Stack:** `packages/cli` (TypeScript), `docs/`, `@cascivo/layouts` (referenced, **not** modified). No new
dependency.

- [ ] **Step 1:** Open with the correction, stated plainly and helpfully: cascivo *does* ship layout primitives and a

- [ ] **Step 2:** Before/after sections mirroring the report's drift: replace `style={{ display:'flex', gap:16,
