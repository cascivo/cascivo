**Goal:** A Usage / Speed Insights surface renders KPI cards (edge requests, function invocations,
bandwidth), an AreaChart (requests/bandwidth over a time range), a BarChart (top paths), and Core Web
Vitals (LCP / CLS / INP) via Meter / Bullet / ProgressCircle — all from seeded deterministic data, with
a time-range selector. `apps/examples/coverage.test.ts` stays green (chart coverage preserved/added).

**Architecture:** `src/sections/Usage.tsx` + `SpeedInsights.tsx`, fed by new seeded series in
`src/data/usage.ts`. Mounted as the project Analytics tab body and as a top-level Usage view (signal
`view`/tab). A time-range signal recomputes the series via `useComputed`.

**Tech stack:** unchanged.

**Files:** `apps/examples/edge/src/data/usage.ts` (new), `usage.test.ts`
