**Goal:** Cascade Edge presents the real Vercel chrome — a team scope switcher, a Cmd+K command menu, a
breadcrumb, and signal-driven navigation between Home / Project / Deployment views (no client router) —
and the **Home view renders a projects grid** of cards backed by a new seeded Vercel domain model, with
search + sort/filter, Skeletons while loading, and an EmptyState when nothing matches.

**Architecture:** New `src/data/` domain model (own fixtures, seeded via kit's `seededRandom`); a new
`src/state.ts` (or similar) holding module-level view-navigation signals; an `AppShell`-based chrome with
a scope switcher in the brand slot and a command menu; a `src/sections/ProjectsGrid.tsx`. Old
`MetricsBar`/`PipelineList`/`EnvironmentGrid` + the kit-based `fixtures.ts` are removed once the new home
renders.

**Tech stack:** unchanged.
