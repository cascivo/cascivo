# Brand & Naming

## Problem

Current name `cascade-ui` collides with an existing UI library, and no matching
domain (`cascade.com` / `.io` / `.dev`, plus `get/use/try` prefixes) is available.
This doc records the rename search.

## Findings

Tested ~130 domains across `.com` / `.io` / `.dev` (2026-06-13). Brutal availability:

- Every clean dictionary word taken (`ripple`, `drift`, `flume`, `quill`, `glint`, `wisp`, `verve`, `vellum`, `stratum`…).
- Every 4–5 letter coinage taken (`layr`, `lume`, `lyra`, `velo`, `fluxe`, `nuvel`, `signl`…).
- Every `get` / `use` / `try` + `cascade` prefix taken.
- Survivors are all longer **coined** words.

## Available domains

| Name        | .com   | .io    | Notes                                             |
| ----------- | ------ | ------ | ------------------------------------------------- |
| **Cascivo** | ✓ free | taken  | Keeps "cascade" lineage, pronounceable, brandable |
| Cascelio    | ✓ free | —      | Cascade-rooted, softer sound                      |
| Cascedo     | ✓ free | —      | Cascade + Latin feel, a bit flat                  |
| Cascven     | ✓ free | —      | Cascade-rooted but awkward consonants             |
| Cascq       | ✓ free | —      | Short but ugly, hard to say                       |
| Florae      | taken  | ✓ free | Botanical, no .com                                |
| Cssui       | taken  | ✓ free | Literal (css+ui), no .com                         |

## Recommendation

**Cascivo** (`cascivo.com`). Only candidate with a free `.com` that stays in the
cascade family — eases the rebrand from `cascade-ui`, reads as one word, says
cleanly. `cascivo.io` is taken; `.com` is the gold standard.

## Open options (not yet explored)

- More TLDs for a chosen name (`.app`, `.sh`, `.ui`, `.design`).
- Variations (`cascivoui.com`, `getcascivo.com`).
- Drop the cascade root entirely — fully invented words in a new direction
  (botanical, geometric, motion).
