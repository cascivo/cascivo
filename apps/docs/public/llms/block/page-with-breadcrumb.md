# PageWithBreadcrumb

A centered content page with a breadcrumb navigation and page header.

## Install

```bash
npx cascade add block/page-with-breadcrumb
```

## Category

`display`

## Props

_No props._

## Examples

### Default

Page with breadcrumb

```tsx
<PageWithBreadcrumb />
```

## Accessibility

- **WCAG level:** 2.1-AA
- **ARIA role:** `generic`

## Dependencies

- `@cascade-ui/react`
- `layout/page-header`
- `layout/center`

## Tags

block, breadcrumb, page, navigation
