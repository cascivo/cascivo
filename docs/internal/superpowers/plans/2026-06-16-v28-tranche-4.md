# v28 — Tranche 4: Preview Panel

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the center-column live preview. A `<style data-cascivo-cre>` element is injected
into `document.head` and its `textContent` is updated whenever `config` changes. All cascivo
components in the preview render inside `data-theme="create-custom"` and re-style automatically as
the CSS changes — no React remounting.

**Architecture:** `PreviewPanel.tsx` uses `useSignalEffect` to manage the `<style>` element
lifecycle. The effect runs `el.textContent = configToCSS(config.value)` on every config change.
Cleanup removes the element when the component unmounts (user navigates away). The `config` signal
is read inside the effect only — never in the render function — so the component itself never
re-renders on config changes. Only the CSS on the page changes.

**Tech Stack:** `@cascivo/core` (useSignalEffect), `@cascivo/react` (Button, Input, Checkbox,
Toggle, Card, CardContent, CardHeader, CardTitle, CardFooter, Badge, Alert, Select), React 18,
`configToCSS` from T2.

### Task 1: Style injection and teardown

This task builds the injection mechanism in isolation (no components rendered yet) so it can be
verified before layering in the showcase markup.

- [ ] **Step 1: Create `apps/landing/src/pages/create/PreviewPanel.tsx`**

  ```tsx
  'use client'
  import { useSignalEffect } from '@cascivo/core'
  import { config } from './store'
  import { configToCSS } from './css-gen'

  export function PreviewPanel() {
    // Inject and maintain a <style> element for the live preview CSS.
    // Reading config.value inside useSignalEffect subscribes to changes.
    // The cleanup callback removes the element on unmount.
    useSignalEffect(() => {
      let el = document.querySelector<HTMLStyleElement>('style[data-cascivo-cre]')
      if (!el) {
        el = document.createElement('style')
        el.dataset.cascivoCre = ''
        document.head.appendChild(el)
      }
      el.textContent = configToCSS(config.value)

      return () => {
        document.querySelector('style[data-cascivo-cre]')?.remove()
      }
    })

    return (
      <div className="preview-frame" data-theme="create-custom">
        <div className="preview-inner">
          {/* Component groups added in Task 2 */}
          <p
            style={{
              color: 'var(--cascivo-color-text-subtle)',
              fontSize: 'var(--cascivo-text-sm)',
            }}
          >
            Components will render here.
          </p>
        </div>
      </div>
    )
  }
  ```
