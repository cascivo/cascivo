# v25 — Tranche 4: ExamplesGallery Carousel

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the static five-card text grid in the "Drive it, don't read about it." section with a full-width carousel that shows one demo at a time: the demo's `desktopLight` screenshot above its long-form description, coverage chips, and a link; auto-advances every 4 s; and has dot indicators + prev/next buttons for manual navigation.

**Architecture:** `ExamplesGallery.tsx` gains `useSignals()`, `useSignal<number>` for active index, and `useSignalEffect` with `setInterval` for auto-advance. The five static `.examples-card` divs are replaced by a single active-demo display with a fixed-aspect-ratio image container (prevents CLS), description text from `demo.description`, coverage chips, and navigation controls. Screenshot paths come from `demo.screenshots.desktopLight` which resolves to `/screenshots/<slug>/light-desktop.png` — these files are already served as static assets. No backend, no new image pipeline.

**Tech Stack:** `useSignal`, `useSignals`, `useSignalEffect` from `@cascivo/core`, `DEMOS` from `pages/examples/data.ts`, `landing.css`.

- [ ] **Step 2: Write the new ExamplesGallery.tsx**

  Replace the contents of `apps/landing/src/sections/ExamplesGallery.tsx` with:

  ```tsx
  'use client'
  import { useSignal, useSignals, useSignalEffect } from '@cascivo/core'
  import { DEMOS } from '../pages/examples/data'

  const INTERVAL_MS = 4000

  export function ExamplesGallery() {
    useSignals()
    const activeIdx = useSignal(0)

    useSignalEffect(() => {
      const id = setInterval(() => {
        activeIdx.value = (activeIdx.value + 1) % DEMOS.length
      }, INTERVAL_MS)
      return () => clearInterval(id)
    })

    const demo = DEMOS[activeIdx.value]

    if (!demo) return null

    function prev() {
      activeIdx.value = (activeIdx.value - 1 + DEMOS.length) % DEMOS.length
    }

    function next() {
      activeIdx.value = (activeIdx.value + 1) % DEMOS.length
    }

    return (
      <section id="examples" className="section examples-gallery" data-reveal>
        <h2>Drive it, don&rsquo;t read about it.</h2>
        <p className="section-sub examples-gallery-sub">
          Five functional mock dashboards, each modelled on a well-known SaaS product. No backend,
          no accounts, no setup — open the URL and play. Collectively they exercise every component
          and every chart in the library.
        </p>

        <div className="examples-carousel">
          {/* Screenshot */}
          <div className="examples-screenshot-frame">
            <img
              key={demo.slug}
              src={demo.screenshots.desktopLight}
              alt={`${demo.name} screenshot`}
              className="examples-screenshot"
              width={1280}
              height={800}
              loading="lazy"
              decoding="async"
            />
          </div>

          {/* Demo info */}
          <div className="examples-carousel-info">
            <div className="examples-card-header">
              <span className="examples-name">{demo.name}</span>
              <span className="examples-feels-like">feels like {demo.feelsLike}</span>
            </div>
            <p className="examples-desc examples-desc--long">{demo.description}</p>
            <ul className="examples-chips" aria-label="Featured components">
              {demo.coverage.map((chip) => (
                <li key={chip} className="examples-chip">
                  {chip}
                </li>
              ))}
            </ul>
            <div className="examples-card-footer">
              <a className="examples-link" href={demo.detailHref}>
                View demo &rarr;
              </a>
              <span className="examples-mock-note">Mock demo &mdash; no real data</span>
            </div>
          </div>

          {/* Navigation */}
          <div className="examples-nav">
            <button
              type="button"
              className="examples-nav-btn"
              aria-label="Previous demo"
              onClick={prev}
            >
              &#8592;
            </button>
            <div className="examples-dots" role="group" aria-label="Select demo">
              {DEMOS.map((d, i) => (
                <button
                  key={d.slug}
                  type="button"
                  className="examples-dot"
                  aria-label={d.name}
                  aria-pressed={activeIdx.value === i}
                  data-state={activeIdx.value === i ? 'active' : undefined}
                  onClick={() => {
                    activeIdx.value = i
                  }}
                />
              ))}
            </div>
            <button
              type="button"
              className="examples-nav-btn"
              aria-label="Next demo"
              onClick={next}
            >
              &#8594;
            </button>
          </div>
        </div>

        <p className="examples-gallery-all">
          <a className="examples-link" href="/examples">
            See all five demos &rarr;
          </a>
        </p>
      </section>
    )
  }
  ```
